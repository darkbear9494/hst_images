This folder is used as the backup of my script on processing the hst image data.

Description of folders:
drz: the scripts to calibrate and drizzle binary AGN images
img: some simple tools dealing with images
iso: obtain the isophote using ellipse in pyraf and draw the 1D profile.
msk: mask the uninteresting sources to obtain the sky or the target sources.
sky: measure the sky using Zhaoyu's method.
trm: trim the image from bad pixels in edges.
rpl: replace the central saturated part with short exposured images.
others: other scripts that maybe useful.
